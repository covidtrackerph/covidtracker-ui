About

This project uses the same data as DOH. This means that i am in no way whatsoever doing manual intervention on the data or running any kind of script, just fetching some data from their source. The only difference with this and DOH tracker is the user interface, it's lighter and mobile friendly so you can easily track the latest on the spread of COVID-19 in the Philippines. This project is fully open-source using Github pages as a host.
